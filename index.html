Git and GitHub Introduction

What is Git?

Git is a distributed version control system that allows multiple developers to collaborate on projects and track changes to source code.

What is GitHub?

GitHub is a web-based platform built on top of Git that provides hosting for software development projects, collaboration features, and more.

Basic Git Commands

git init - Initialize a new Git repository in the current directory.

Example: git init

git clone [repository_url] - Clone a repository into a new directory.

Example: git clone https://github.com/user/repo.git

git add [file] - Add a file to the staging area.

Example: git add index.html

git commit -m "[message]" - Commit changes to the repository with a descriptive message.

Example: git commit -m "Add initial HTML structure"

git push - Push local changes to a remote repository.

Example: git push origin main

git pull - Fetch and merge changes from a remote repository.

Example: git pull origin main

git status - Show the status of working directory and staging area.

Example: git status
